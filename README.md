# Scene.it - AI Visual Discovery Platform

**See what you imagine. Find what you feel.**

Scene.it is a professional visual intelligence tool built for designers, art directors, and creatives who think in moods, not keywords.

## Features

- **Natural Language Search** - "cozy coffee shop with warm lighting"
- **AI-Powered Intent Parsing** - GPT-4 extracts mood, colors, style
- **Multi-Platform Discovery** - Pinterest, Instagram, Unsplash
- **Minimal UI** - One search bar, one save button

## Tech Stack

Next.js 16 | TypeScript | Tailwind | Supabase | OpenAI | Apify

## Quick Start

```bash
npm install
cp .env.example .env.local
# Add your API keys
npm run dev
```

## Environment Variables

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
OPENAI_API_KEY=
APIFY_API_TOKEN=
UNSPLASH_ACCESS_KEY=
```
